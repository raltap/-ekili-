# Çekiliş Pro v2.0

Modern, hızlı ve görsel efektlerle zenginleştirilmiş profesyonel çekiliş aracı.

## Özellikler
- **Sınırsız Katılımcı:** İstediğiniz kadar isim ekleyebilirsiniz.
- **Slot Machine Animasyonu:** Gerçekçi ve heyecan verici sonuç animasyonu.
- **Ses Efektleri:** Çekiliş anında ve kazanma anında özel sesler (Açılıp kapatılabilir).
- **Konfeti Patlaması:** Kazanan açıklandığında görsel şölen.
- **Yedek Çekiliş:** Ana kazananların yanı sıra istediğiniz sayıda yedek belirleme.
- **Detaylı Log:** Yapılan tüm çekilişlerin kaydı.
- **Otomatik Büyük Harf:** Tüm isimler otomatik olarak büyük harfe çevrilir.

## Nasıl Kullanılır?
1. Katılımcıları her satıra bir isim gelecek şekilde sol taraftaki alana yapıştırın.
2. Ayarlar kısmından kaç kazanan ve kaç yedek olacağını seçin.
3. Çekiliş süresini belirleyin (1-10 saniye).
4. "ÇEKİLİŞİ BAŞLAT" butonuna basın.

## Geliştiriciler İçin
Bu proje React + Vite + Tailwind CSS ve motion kullanılarak geliştirilmiştir.

### Kurulum
```bash
npm install
npm run dev
```

### Yayınlama (Build)
```bash
npm run build
```
Cloud Run veya GitHub Pages gibi ortamlarda otomatik olarak yayına alınabilir.
